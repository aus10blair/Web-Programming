1. To have quality control use html and css to create a functioning site for the school
  2. The use of an external style sheet would setup the website to be uniform and function the same for each teachers' page, This reduces the amount of navigation time amongst teachers and students
  3. Possible html tags include
    • "title" to give each page a name
    • "ol" to list out assignments on each page
    • "background-color" to make the page look better, maybe reference the schools colors
    • "style" to make an overarching theme for the school website
    • "a href" to link to documents such as sylabi, assignments, etc
    • "table" to initialize a table for schedules and other things
    • "p" for any paragraphs faculty and staff may need to put on their pages
    • "h1" for titles of pages such as teachers names, etc
    • "font-family" to make sure text is displayed uniformly across the site
    • "h2" for information that should be emphasized
